# 🚇 Smart Metro Data Analytics and Visualization

## 📌 Project Overview
This project analyzes metro passenger data to understand usage patterns and improve operational efficiency. It helps identify peak hours, busiest routes, and passenger flow trends.

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Data Visualization (Matplotlib / Charts)
- Django (if used)
- SQLite

---

## 📊 Features
- Data cleaning and preprocessing
- Analysis of metro passenger data
- Identification of peak hours and busiest stations
- Visualization of trends using charts and graphs

---

## 📁 Project Structure
- `dataset/` → Contains metro data
- `analytics/` → Data analysis code
- `templates/` → Frontend templates
- `static/` → CSS/JS files
- `manage.py` → Django main file
